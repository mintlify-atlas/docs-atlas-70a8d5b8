---
title: Lesson
description: TypeScript types for lesson structure and scene control
---

The `ParsedLesson` type represents a fully parsed lesson with narration, visualizations, and scene animations.

## ParsedLesson

<ParamField path="title" type="string" required>
  Lesson title
</ParamField>

<ParamField path="steps" type="readonly LessonStep[]" required>
  Sequence of lesson steps (one per H1 section)
</ParamField>

<ParamField path="diagnostics" type="readonly LessonDiagnostic[]" required>
  Parser warnings and errors
</ParamField>

## LessonStep

A single section of a lesson.

<ParamField path="id" type="string" required>
  Unique step identifier
</ParamField>

<ParamField path="title" type="string" required>
  Step title from markdown H1
</ParamField>

<ParamField path="narration" type="readonly NarrationBlock[]" required>
  Narration paragraphs with embedded triggers
</ParamField>

<ParamField path="blocks" type="ReadonlyMap<string, VisualizationState>" required>
  Named visualization blocks defined in this step
</ParamField>

<ParamField path="scenes" type="readonly SceneState[]" required>
  Computed scene states for playback
</ParamField>

## NarrationBlock

<ParamField path="text" type="string" required>
  Plain text narration for TTS
</ParamField>

<ParamField path="triggers" type="readonly Trigger[]" required>
  Scene control triggers embedded in narration
</ParamField>

## Trigger

<ParamField path="wordIndex" type="number" required>
  Zero-based word index in narration block
</ParamField>

<ParamField path="text" type="string" required>
  Original trigger syntax from markdown
</ParamField>

<ParamField path="action" type="TriggerVerb" required>
  Parsed trigger action. See [TriggerVerb](#triggerverb) for variants.
</ParamField>

</Expandable>

<Expandable title="hide">

<Expandable title="hide-group">

## AnimationOverride

Discriminated union for animation configuration.

<Expandable title="default">
  <ParamField path="kind" type="'default'" required>
    Use default animation settings
  </ParamField>
</Expandable>

<Expandable title="custom">
  <ParamField path="kind" type="'custom'" required />
  <ParamField path="effect" type="AnimationEffect" required>
    One of: `'fade'` | `'slide'` | `'slide-up'` | `'grow'` | `'typewriter'` | `'none'`
  </ParamField>
  <ParamField path="durationS" type="number" required>
    Animation duration in seconds
  </ParamField>
  <ParamField path="easing" type="EasingKind" required>
    One of: `'ease-out'` | `'ease-in-out'` | `'spring'` | `'linear'` | `'reveal'` | `'emphasis'` | `'handoff'`
  </ParamField>
</Expandable>

## SceneState

Represents the computed state of a single animation frame.

<ParamField path="slots" type="readonly VisualizationState[]" required>
  Active visualizations in this scene
</ParamField>

<ParamField path="transition" type="TransitionKind" required>
  Scene transition effect
</ParamField>

<ParamField path="enterEffects" type="readonly SlotEnterEffect[]" required>
  Enter animations for appearing slots
</ParamField>

<ParamField path="exitEffects" type="readonly SlotEnterEffect[]" required>
  Exit animations for disappearing slots
</ParamField>

<ParamField path="epoch" type="number" required>
  Scene epoch (incremented on clear)
</ParamField>

<ParamField path="focus" type="string" required>
  Currently focused region (empty string if none)
</ParamField>

<ParamField path="flow" type="string" required>
  Active flow animation target
</ParamField>

<ParamField path="pulse" type="string" required>
  Pulsing element target
</ParamField>

<ParamField path="trace" type="string" required>
  Traced path target
</ParamField>

<ParamField path="transformFrom" type="readonly { readonly to: string; readonly from: string }[]" required>
  Active transform mappings
</ParamField>

<ParamField path="annotations" type="readonly SceneAnnotation[]" required>
  Scene annotations
</ParamField>

<ParamField path="zoom" type="{ readonly scale: number; readonly target: string }" required>
  Zoom state
</ParamField>

<ParamField path="pan" type="string" required>
  Pan target
</ParamField>

<ParamField path="draw" type="string" required>
  Drawing animation target
</ParamField>
